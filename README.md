# 📱 Social App - Modern Angular v21

A high-performance Social Media Application built with **Angular 21**, featuring Server-Side Rendering (SSR) for optimal SEO and performance.

## 🚀 Live Demo
[🔗 View Live Project on Netlify](رابط_موقعك_على_نتليفاي_هنا)

## ✨ Key Features
* **Modern Angular (v21):** Developed using Standalone Components and Signals.
* **Server-Side Rendering (SSR):** Optimized for SEO and fast initial page loads.
* **Authentication:** Fully functional Login and Sign-up system.
* **Dynamic Feed:** CRUD operations for Posts and Comments.
* **Responsive UI:** Crafted with **Tailwind CSS** & **Flowbite** (Facebook-inspired UI).
* **Notifications:** Real-time-like notification center and modals.

## 🛠️ Tech Stack
* **Frontend:** Angular 21 (Standalone, SSR, Signals)
* **Styling:** Tailwind CSS, Flowbite
* **Icons:** FontAwesome
* **Hosting:** Netlify (with CI/CD Pipeline)
